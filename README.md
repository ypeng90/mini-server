# Minimal HTTP Server

## Main Features

- Concurrency supported
- Methods implemented: `GET`, `HEAD`, `POST`
- Static contents supported: `html`, `gif`, `png`, `jpg`, `mp4`, `plain text`
- Dynamic contents supported: `CGI`
